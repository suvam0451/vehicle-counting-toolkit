### Inputnew folder

The `inputnew` folders has id tracked trajectories of vehicles, in the following format

```json
[
  {
    "vehicle_id": 451,
    "frame_count": 226,
    "class_id": 5,
    "objects": [
      {
        "center_x": 0.453294,
        "center_y": 0.176482,
        "confidence": 0
      },
      {
        "center_x": 0.452867,
        "center_y": 0.176561,
        "confidence": 0
      }
    ]
  }
]
```
